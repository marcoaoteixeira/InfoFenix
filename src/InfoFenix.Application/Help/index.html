Pesquisar

Termos simples

Você pode realizar uma pesquisa, pelos documentos indexados, utilizando o menu "Pesquisar".

Ao clicar no menu, a tela de pesquisa será aberta. Nessa tela, você irá realizar a pesquisa. Para isso, digite no campo "Pesquisar por:" o termo pelo qual deseja buscar. Depois, clique no botão "Pesquisar".

Caso o motor de pesquisa encontre resultados, a quantidade de itens encontrados será exibida na coluna "Documentos encontrados". Para acessar esses documentos, dê um duplo clique na linha do diretório de documentos desejado.

Caso o motor de pesquisa não encontre resultados, uma mensagem de alerta será exibido no canto inferior esquerdo da tela. Juntamente, a coluna "Documentos encontrados" exibirá o valor 0 (zero).

Utilizando caracter chave (*)

É possível realizar uma pesquisa por termos aproximados. Para isso, utilize o caracter "*" no início, ou no fim, do termo utilizado para a pesquisa. Neste exemplo, obteve-se resultados para a pesquisa por "adv*". Esse termo poderia se referir à "advocacia", "advogado", "adversário", etc. Vale lembrar que o caracter "*" pode ser utilizando no começo do termo também, como no exemplo "*der", que poderia se referir ao termo "poder", "arder", "ceder", etc.

Ignorando termos

É possível realizar uma pesquisa e solicitar que certos termos sejam ignorados quando o resultado for retornado. Para tal, basta adicionar o sinal "-" antes do termo a ser pesquisado. Por exemplo: "hotel -reserva". Uma pesquisa por esses termos traria documentos que contem a palavra "hotel", porém não contenham a palavra "reserva", no mesmo documento.
